# Welcome to EpiMetal documentation

## Introduction
EpiMetal is an interactive single-page web application for real-time visualization and analysis of data.

## How to use
Please see the [User guide](userguide.md) on how to use this application.

## Installation
Want to try EpiMetal on your data and install it on your own server? EpiMetal can be installed on various platforms by using Docker files provided. For instructions on how to deploy a custom instance of EpiMetal, see the [installation instructions](installation.md).

## Source code
EpiMetal is an open-source project and its source code is freely available at the [Github project page](http://github.com/amergin/epimetal).

## Issues
If you feel you have discovered a bug within this software, please report it by opening an issue on the [Github project page](http://github.com/amergin/epimetal/issues).
